Seven Stars Bakery Redesign

Redesigning the website of my favorite Providence bakery <3

Current Webpage

I chose this website because I love Seven Stars's bread and pastries, but I am not in love with their website ://

While examining the site, I found a few issues involving its usability, learnability, memorability, and accessibility (as detailed below). I used the WebAIM WAVE tool to help find accessibility issues, and I found one main issue worth noting, which was the very low contrast for the heading text. The background image is quite saturated, so it is difficult to read the title when first opening the page. The tool was useful, but many of the warnings it picked up did not seem to hinder accessibility to a great extent (such as noscript tags).

Key Issues

  1. There are no dropdown icons for the menu bar, making it hard to determine which categories have additional items (usability)
  2. There is no indication whether the top categories are linked (learnability)
  3. The navigation bar goes away when you scroll down the page (usability)
  4. Some panels in the second section don't have labels, so it is hard to determine what they link to (learnability)
  5. Sections with the panels, large image for baking process, and coffee info seem random and make it difficult to remember what is on the home page (memorability)
  6. Locations are refered to as bakeries and cafes, which could be confusing (learnability)
  7. There is no map to visualize where the locations are (usability)
  8. It is unclear whether the bottom links in the footer are clickable even though they are (learnability)
  9. Very low contrast for main heading (accessibility)

Changes Made

  1. Drop down indicators added to show clearly which menu items can be expanded
  2. Menu items will change on hover
  3. Fixed navigation bar
  4. All panels have labels added, and only relevant panels were kept
  5. Simplified home screen by removing uneccessary sections
  6. Changed section title to avoid ambiguity
  7. Added a map! (wanted to make interactive, but its just an image placeholder for now)
  8. Underlined links in footer to make them clearer
  9. Changed and desaturated heading image to increase contrast



Style Guide

Using the lo-fi prototypes, I developed a style guide to make hi-fi prototypes of the sites on Figma. This details the colors as well as the text, button, and navigation bar styles to be used in the final site, including how different components react to being hovered or clicked.
